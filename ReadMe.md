# Wake & Bake - Website

A premium, single-page responsive website for **Wake & Bake**, a restaurant specializing in Biryani, Momos, and Fast Food. This project showcases a modern, appetizing design tailored for a local food business.

## 🥘 About the Project

**Wake & Bake** is designed to be a digital storefront for a restaurant, offering customers an easy way to browse the menu and contact the business. The design focuses on a "rustic yet premium" aesthetic, using a warm color palette (Terracotta/Orange) to stimulate appetite and convey quality.

**Key Highlights:**

- **Visual Appeal:** High-quality imagery, glassmorphism effects, and smooth animations.
- **User Experience:** Clear navigation, sticky header, and mobile-optimized layout.
- **Information Accessibility:** prominent contact information and categorized menu sections.

## ✨ Features

- **Responsive Design:** Fully fluid layout that adapts perfectly from mobile phones to desktop screens.
- **Interactive Menu:** Categorized sections for Biryani, Momo, Chinese, and Rolls for easy browsing.
- **Glassmorphism Header:** A modern, translucent sticky header for seamless navigation.
- **Call-to-Action:** Floating "Contact Now" buttons and easy access to phone numbers.
- **Custom Styling:** tailored styling using Tailwind CSS configuration for specific brand colors and fonts.

## 🛠 Tech Stack

- **HTML5:** Semantic structure.
- **Tailwind CSS (via CDN):** Utility-first CSS framework for rapid and responsive styling.
- **JavaScript:** Minimal vanilla JS (embedded) for specific interactions if needed (currently purely CSS/HTML focused).
- **FontAwesome:** For vector icons (phone, location, menu icons).
- **Google Fonts:**
    - _Inter_ (Sans-serif) for readability.
    - _Playfair Display_ (Serif) for headings and elegance.

## 🚀 Getting Started

Since this project uses static HTML and CDN-based libraries, no build step is required!

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/wake-and-bake-website.git
    ```
2.  **Navigate to the project folder:**
    ```bash
    cd wake-and-bake-website
    ```
3.  **Open `index.html`:**
    Simply double-click `index.html` to open it in your preferred web browser. Alternatively, you can use a live server extension (like Live Server in VS Code) for a better development experience.
